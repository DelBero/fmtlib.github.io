C++ Format

Small, safe and fast formatting library for C++

Download

Overview

C++ Format (cppformat) is an open-source formatting library for C++. It can be used as a type-safe alternative to printf or as a fast alternative to IOStreams.

Features

Format API

The replacement-based Format API provides a type-safe alternative to printf, sprintf and friends while being as fast or even faster. The format string syntax is similar to the one used by str.format in Python: 

Format("The answer is {}", 42);

The Format API supports positional arguments useful for localization.

Write API

The concatenation-based Write API (experimental) provides a fast stateless alternative to IOStreams: 

Writer w;
w << "The answer in hexadecimal is " << hex(42);

Safety

The library is fully type safe, errors in format strings are reported using exceptions, automatic memory management prevents buffer overflow errors. For example, the code 

Format("The answer is {:d}", "forty-two");

throws a FormatError exception with description "unknown format code 'd' for string", because the argument "forty-two" is a string while the format code d only applies to integers.

Portability

C++ Format is highly portable. Here is an incomplete list of operating systems and compilers where it has been tested and known to work:

  • 64-bit (amd64) GNU/Linux with GCC 4.4.3, 4.6.3, 4.7.2, 4.8.1 and Intel C++ Compiler (ICC) 14.0.2
  • 32-bit (i386) GNU/Linux with GCC 4.4.3, 4.6.3
  • Mac OS X with GCC 4.2.1 and Clang 4.2, 5.1.0
  • 64-bit Windows with Visual C++ 2010
  • 32-bit Windows with Visual C++ 2010

Although the library uses C++11 features if available, it also works with legacy compilers that only support older standards. For example, if the compiler doesn't support initializer lists, you can use << operator to feed arbitrary arguments into the Format function: 

Format("The answer is {}") << 42;

Ease of Use

C++ Format has small self-contained code base consisting of a single header file and a single source file and no external dependencies. A permissive BSD license allows using the library both in open-source and commercial projects.
